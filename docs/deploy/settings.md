# 配置中心

后台侧栏 **配置中心** 集中管理运行所需配置。页面为单列流：Cloudflare + 邮件维护。

首次部署后若尚未连接 Cloudflare，侧栏「配置中心」会有轻量提示；进入后可通过 **开始配置** 填写接口令牌。

## Cloudflare

通过接口令牌连接 Cloudflare 账号后，可配置：

| 项 | 说明 |
| --- | --- |
| 账号 ID | Cloudflare Account ID（令牌校验后可选择） |
| Worker | 当前 DoneMail 使用的 Worker 名称 |
| 后台入口 | 管理后台访问入口；可选「任意」 |
| 共享入口 | 共享链接使用的入口；默认跟随后台入口，也可单独指定 |

令牌用于读取 Zone、检查 Email Routing、配置 Catch-all、读取 Worker 路由等。保存后即可在 **域名管理** 中添加主域与子域。

未配置令牌时，添加主域会提示先到配置中心完成连接。

## 邮件维护

| 项 | 说明 |
| --- | --- |
| 定时清理 | 开启后，定时任务按保留天数清理收件箱邮件 |
| 邮件保留天数 | **最少 1 天**，最多 3650 天；修改后自动保存 |

清理时会删除数据库中的邮件记录；R2 中的附件对象会尽量同步删除，失败时会进入后台回收队列。

## 访问保护（内置）

限流为系统内置，**暂时不能通过配置修改**：

| 类型 | 上限（每 IP / 小时） |
| --- | --- |
| 初始化管理员 Key | 20 |
| 登录失败 | 60 |
| 公开接口鉴权失败 | 120 |
| 共享访问 | 2000 |

计数保存在 KV。如需调整，只能修改代码中的内置常量后重新部署。

## 其它入口

| 能力 | 位置 |
| --- | --- |
| 修改管理员 Key | 右上角用户菜单 |
| 检查更新 | 右上角用户菜单；有新版本时高亮；在自己的 Fork 上 Sync fork 后自动部署 |
| 异常记录 | 侧栏「异常记录」 |
